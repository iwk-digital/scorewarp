ScoreWarp Help

ScoreWarp is a toolkit for the analysis and visualization of music performance data. It allows you to warp musical scores and visualize the results in an interactive way.

How to Use ScoreWarp

  1. Load a MEI score file and one or more MAPS files that reference the ids of the MEI file.
  2. Select a demo piece with performances or upload your own files.
  3. Click on the "Warp" button to warp the score.
  4. Use the "Warp notes inside chords" button to warp individual notes within chords.
  5. Reload data if needed using the "Reload data" button.
  6. Download the warped score as an SVG file using the "Download SVG" button.
  7. Toggle red lines to visualize the warping effect.
  8. View the warped score and performance time in the presentation panel.
  9. Use the "Drag your files" area to upload your own MEI and MAPS files.
  10. Check the task list for the status of your uploads.
  11. Click on the "Toggle red lines" button to visualize the warping effect.
  12. Click on the "Download" link to save the warped score to your browser's download folder.
  13. Check the copyright information at the bottom of the page.
